# 在线相册管理系统



## 1、项目介绍

在线相册管理系统中，功能简单，管理员可以管理文件，上传图片等功能。游客用户可以查看相册图片，评论图片等，适合学习技术


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可）
- maven项目：否 
- 硬件环境：Windows 或者 Mac OS
